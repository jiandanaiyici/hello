- [世界各国数据](https://mledoze.github.io/countries/): 包含 `JSON CSV and XML and Yaml`
- [中国地图](https://gw.alipayobjects.com/os/basement_prod/32e1f3ab-8588-46cb-8a47-75afb692117d.json)

### 数据转换

- [不同坐标系之间的转换](https://github.com/hujiulong/gcoord): 包含 高德 和 百度等地图之间的经纬度相互转换, [具体参考](https://github.com/hujiulong/gcoord/wiki/%E5%9C%B0%E7%90%86%E5%9D%90%E6%A0%87%E7%B3%BB)

### 在线工具

- [http://geojson.io/](http://geojson.io/) 可以在线查看，绘制，修改 GeoJSON 数据
- [https://mapshaper.org/](https://mapshaper.org/) 可以查看较大的 geojson，还能够简化 GeoJSON 数据
- [随机生成经纬度](http://www.geomidpoint.com/random/calculation.html)
- [随机生成 mock 数据](https://randomuser.me/) [randomapi](https://randomapi.com/)
- [地图上随机点击获取经纬度](https://3geonames.org/)
