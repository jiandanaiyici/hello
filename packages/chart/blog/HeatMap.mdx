## 背景

> 在某一天收到了一个信息, 要开发一个基于 [Antv L7](https://l7.antv.vision/zh/)上层开发一个热力图表, 首先开发过一些图表, 地图相关的还是第一次开发, 然后就很兴奋的去看官方文档, 找到对应的图表类型以及各种 API 就着急于开发过程中新接触一个东西时总会是有不同的问题出现慢慢的习惯了记录一些东西, 也算是一种总结吧

## 准备

- 查看对应的基础文档, 了解简单的需求, 首先要做的就是实现一个基础版本的地图, 了解到使用到的是高德地图, 需要查看下[高德 JS API](https://lbs.amap.com/api/javascript-api/reference/layer#m_AMap.Heatmap) 的相关信息, 在上一篇中已经大概了解过了基于`React`版本的组件实现, 那么这次就使用 `React` 组件进行实践
