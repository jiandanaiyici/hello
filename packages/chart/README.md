# Website

[Docusaurus 2](https://v2.docusaurus.io/): `React` 静态站点生成器

```bash
# 安装
npm install

# 启动
npm start

# 输出 build 文件夹
yarn build

# 部署 surge
npm run surge
```

### 部署

```
$ GIT_USER=<Your GitHub username> USE_SSH=true yarn deploy
```
