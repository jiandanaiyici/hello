> 创建对象的一些方式

### assign

> 定义: 通过一个或多个对象来创建一个新的对象, 当有包含相同属性名时后一个属性值覆盖前一个属性值,
> 只会拷贝对象自身的且可枚举的属性到目标对象, 会调用相关的 getter 和 setter, 因此分配属性, 而不仅仅是赋值或定义新的属性, 如果合并源包含 getter,
> 这可能使其不适合将新属性合并到原型中

> 注意: 如果源对象值为 null 或 undefined 时不会抛出错误
