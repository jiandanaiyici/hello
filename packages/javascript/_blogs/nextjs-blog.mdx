- [x] 首先按照以上文档创建好了基于 NextJs 搭建的 Typescript 环境之后
- [x] 安装 antd
- [x] 此处未找到解决方法(按照官网说法是现在已经自动支持动态加载不需要手动配置), 暂时使用基于 V3.x 的办法引入动态加载
- [x] 配置 .babelrc 文件, 将 postcss.config.js 文件中的 plugins 注释掉

```bash
yarn add antd babel-plugin-import less @zeit/next-less less-vars-to-js null-loader
```

`.babelrc`

```json
{
  "presets": ["next/babel"],
  "plugins": [
    [
      "import",
      {
        "libraryName": "antd",
        "style": true
      }
    ]
  ]
}
```
