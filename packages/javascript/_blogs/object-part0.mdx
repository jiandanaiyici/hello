### 简述

> 在 JS 中, 万物皆是对象因为所有的不管是内置的构造函数还是创建出来的对象都是基于继承或者 Object 的实例,
> 一个对象是零个或多个属性的集合

### 基本类型

> 一个原始值直接代表语言实现的最低层次的数据

- Undefined
- Null
- Boolean
- Number
- String
- Symbol

---

构造函数: 创建和初始化对象的函数对象, 构造器的 prototype 属性值是用来实现继承和共享的一个原型对象
